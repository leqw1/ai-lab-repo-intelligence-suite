# ai-lab-repo-intelligence-suite

Ai Lab Repo Intelligence Suite is a production-oriented product generated by Autonomous AI Lab.

This repository is intentionally larger than a demo scaffold. It contains a
backend API, static operator dashboard, domain services, tests, security notes,
operations docs, and a Docker-ready release path.

## Product Surface

- FastAPI backend with health, project, agent, risk, and roadmap endpoints.
- Static web dashboard served from the backend.
- Domain services for a company-style AI team workflow.
- CLI entrypoint for local checks and scripted operation.
- Unit tests for the core planning and risk logic.
- Dockerfile and docker-compose file for reproducible local deployment.

## Local Run

```bash
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## Docker Run

```bash
docker compose up --build
```

## Production Readiness

The generated product starts in a safe local mode. Add real integrations only
through `.env.example`, never through committed secrets.
